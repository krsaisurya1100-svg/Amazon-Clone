

# Amazon Clone

This is a front-end clone of Amazon’s homepage that I built to improve my UI/UX and layout skills. The goal was to recreate a real-world e-commerce interface as closely as possible and understand how large websites structure their pages.

## What I built

* A full homepage layout inspired by Amazon
* Navigation bar with logo, location, search bar, and menu
* Multiple product/category sections (books, fashion, electronics, etc.)
* Deals and “best sellers” sections
* Grid-based product layout
* Reusable card-style components
* Basic responsive design for different screen sizes

## Features

* Clean and structured UI similar to Amazon
* Proper spacing, alignment, and positioning
* Section-based design for scalability
* Hover effects and simple interactions
* Organized layout for readability

## Tech stack

* **HTML** → structure
* **CSS** → styling, layout (flexbox/grid)
* **JavaScript** → minor interactivity
